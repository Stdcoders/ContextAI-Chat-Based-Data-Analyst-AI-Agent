# Context AI - Data Analyst AI Agent 

Tech Stack - Langchain, Langgraph, Pandas, ReportLab, Plotly

LLMs used - Groq : GPT-OSS-120, Deepseek-R2, Gemma-2-9b-it

Agent Flow - 
Data Steward : Ingestion -> Data Janitor : Cleaning and analysis -> Data Explorer : Question Generation -> Insights Generator : Provides visualizations and insights -> Report Generator : Creates a PDF report for the ease of user.
